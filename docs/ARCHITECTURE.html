Data Flow

  1. INPUT   : capricorn remember / MCP / OSB Bridge
  2. STORE   : SQLite + FTS5 + Vector + Vault mirror
  3. ENRICH  : Forge extracts insight -> scene -> persona
  4. COMPOUND: Dream accumulates evidence, confidence grows
  5. VALIDATE: HyperTune + HaluGard G2-G4 filter noise
  6. OUTPUT  : recall / context / persona-core.md / prompt-ops winner
